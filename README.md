# 수능 영어 단어장 프로젝트 (My Vocabulary)

수능 필수 영단어 학습 및 암기 상태 관리를 위한 React 기반 웹 애플리케이션입니다. 5,000개 이상의 기출 단어와 자동 학습 기능을 제공합니다.

## 🚀 주요 기능
- **방대한 단어 데이터**: 엑셀(지천명영어) 및 PDF(수능/평가원 필수) 파일에서 추출한 5,000개 이상의 필수 수능 영단어 탑재
- **자동 학습 모드 (Study Mode)**: 
  - 전체 화면 학습 UI 및 자동 넘김(Auto-play) 기능
  - 넘김 간격 설정 가능 (1초~10초)
  - 학습 중 '암기 완료' 체크 시 즉시 다음 단어로 이동
- **카테고리별 필터링**:
  - 지천명영어, 수능필수, 평가원, 분위기/심경 등 출처별 단어 모아보기
  - 학습 중, 암기 완료 상태별 필터링
- **발음 및 음성 지원**:
  - 3,600개 이상의 단어에 대한 IPA 발음기호 표시
  - Web Speech API를 활용한 원어민 발음(TTS) 재생 기능
- **페이지네이션**: 한 페이지당 단어 수(10, 20, 30, 50개) 선택 및 페이지 이동 기능
- **데이터 보존**: `localStorage`를 통해 학습 진척도(암기 완료 상태) 자동 저장 및 초기화 기능

## 🛠 기술 스택
- **Frontend**: React 19 (Vite)
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Data Parsing**: `xlsx`, `pdf-parse` (데이터 추출용)

## 📂 프로젝트 구조
- `src/App.jsx`: 메인 애플리케이션 로직, 상태 관리 및 학습 모드 UI
- `src/data/words.js`: 5,000여 개의 영단어 데이터베이스
- `convert_excel.js`, `process_pdfs.js`: 엑셀/PDF 데이터 추출 및 가공 스크립트
- `tag_categories.js`, `add_ipa.js`: 카테고리 태깅 및 발음기호 추가 자동화 스크립트

## 📝 개발 및 데이터 업데이트 과정
1. **데이터 통합**: 지천명영어 엑셀 파일 및 수능/평가원 필수 PDF 리스트를 분석하여 중복을 제거하고 단어 데이터를 통합했습니다.
2. **IPA 추가**: 오픈 소스 IPA 사전을 활용해 약 3,600개 단어에 정밀한 발음기호를 자동으로 매칭했습니다.
3. **UI/UX 개선**: 대량의 단어를 효율적으로 학습할 수 있도록 페이지네이션과 자동 재생 학습 모드를 구현했습니다.
4. **품질 검증**: ESLint 및 빌드 테스트를 통해 코드 안정성을 확보했습니다.

## ⚠️ 참고 사항
- 본 프로젝트는 교육용 목적으로 제작되었으며, 학습 진척도는 브라우저의 `localStorage`에 저장되므로 브라우저를 변경하거나 시크릿 모드를 사용하면 초기화될 수 있습니다.
- 음성 출력(TTS) 기능은 브라우저의 시스템 설정을 따릅니다.
